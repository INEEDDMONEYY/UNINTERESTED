# Release Notes v2.1.0

## Major Updates

- **Dual Badge Support:** Admins can now grant both the blue verified badge and paid promo badge to users at the same time. Both badges will display on posts and user profiles if present.
- **Unlimited Media Uploads:** There are no longer any limits on the number or size of images/videos you can upload when creating a post (subject to backend/Cloudinary limits).
- **Improved Badge Display:** Post cards and user profile headers now show both badges if a user has both verified and promo status.
- **Contact & Support:** The Contact page and footer have been improved for easier support access.
- **Manual Cash App Confirmation:** Payments via Cash App must be confirmed manually by admins; there is no automatic integration.
- **Bug Fixes:** Various fixes for badge propagation, display, and admin tools.

Thank you for using onex! If you have questions or feedback, please use the Contact page or reach out to support.
